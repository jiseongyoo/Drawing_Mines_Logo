=> How to run the program

1. In a new terminal:
	$ roscore

2. In a new terminal:
	$ rosrun turtlesim turtlesim_node

3. In a new terminal:
	$ cd ~/mines_yoo_jiseong
	$ source ./devel/setup.bash
	$ rosrun mines_yoo_jiseong mines_yoo_jiseong


=> How to compile the .cpp

1. In a new terminal:
	$ cd ~/mines_yoo_jiseong
	$ catkin_make
